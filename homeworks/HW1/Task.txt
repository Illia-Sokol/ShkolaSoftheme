1. Скачайте и  установите MS Visual Studio.
2. Зарегистрируйтесь и создайте Ваш персональных Git аккаунт.
3. Скачайте и установите программу для разобы с Git системой контроля версий (Tortoise Git или другую).
4. Создайте публичный репозиторий под вашим  Git аккаунтом с именем ShkolaSoftheme.
5. Создайте папку CSharpHW в вашем репозитории.
6. Создайте папку с номером лекции внутри папки CSharpHW, поместите туда solution с выполненным домашним заданием. (ShkolaSoftheme/CSharpHW/1/HW1.sln)
7. Создайте консольное приложение и вывидите ваше имя желтым цветом в консоль.
8. После выволнения домашнего задания убедитесь что изменения доступны из под вашего публичного профиля Git.

Аналогичным образом выполняйте последующие задания.
